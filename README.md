# Investment-Simulation
Investment simulator that models hundreds of randomized market scenarios to show your real odds of reaching financial independence — vanilla JS, hand-drawn canvas charts, zero dependencies.

Files created:

- `index.html` — main HTML file; links to `styles.css` and `script.js`
- `styles.css` — extracted styling
- `script.js` — extracted JavaScript

To run locally, open `index.html` in a browser, or serve the folder with a simple HTTP server:

```bash
# from the folder containing these files
open index.html            # macOS: opens the file in the default browser
# or serve via Python
python3 -m http.server 8000
# then visit http://localhost:8000
```
